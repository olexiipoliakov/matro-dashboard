# Дашборд эффективности рекламы Matro

Статический дашборд для GitHub Pages. Две вкладки: «Кампании» (расход, обращения, цена обращения) и
«Эффективность баннеров» (лиды, цена за лид, CTR с сортировкой). Данные из Meta Ads.

## Файлы
- `index.html` — дашборд (открывается на GitHub Pages)
- `data.json` — данные из Meta API: расход по дням и кампаниям, обращения, цена обращения, баннеры (лиды/цена за лид/CTR)
- `fetch_meta.py` — скрипт выгрузки из Meta API в data.json

## Деплой на GitHub Pages
1. Создай репозиторий, загрузи `index.html`, `data.json` (и `fetch_meta.py`).
2. Settings → Pages → ветка `main`, папка `/root` → Save.
3. Через минуту: `https://ТВОЙ_ЛОГИН.github.io/НАЗВАНИЕ/` — открывается с любого устройства.

## Обновление данных
```
pip install requests
set META_ACCESS_TOKEN=твой_токен
set META_ACCOUNT_ID=act_XXXXXXXXXX
python fetch_meta.py
```
Скрипт создаст свежий `data.json` — закоммить его в репозиторий
(можно повесить на планировщик Windows, напр. раз в день).

## Локальная проверка
Двойной клик по index.html не сработает (браузер блокирует чтение JSON
через file://). Запусти в папке проекта:
```
python -m http.server
```
и открой http://localhost:8000

## Настройка
- **Что считать обращением** — список `LEAD_ACTIONS` в `fetch_meta.py`.
  При запуске скрипт печатает, какие типы действий реально прислал Meta.
- **Подсветка цены обращения** (дёшево/средне/дорого) считается относительно
  средней по кабинету — порог в `index.html`, функция `cplClass`.
